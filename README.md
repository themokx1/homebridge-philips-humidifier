# homebridge-philips-humi

A Homebridge plugin for controlling Philips HU1509 / HU1510 humidifiers using the local CoAP+ API through `aioairctrl`.

## Features
✔ On/Off  
✔ Current humidity  
✔ Target humidity  
✔ Fully local control (no cloud)  
✔ Works with Philips HU1509 / HU1510  
✔ Homebridge UI config support

## Requirements
- Python installed
- `aioairctrl` installed:
  ```bash
  pip install aioairctrl
